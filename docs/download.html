Download & Installation

Welcome to the download section of Kurgan MUD. This guide explains how to obtain, compile, and run the MUD server on a modern Linux system. The current codebase has been tested with GCC 11.4.0 on Ubuntu 22.04 LTS.

To get started, you can clone the repository using git: 

git clone https://github.com/ozgyilmaz/kurgan-mud.git
    cd kurgan-mud/src
    make

The compiled binary will be named rom and placed in the src directory. Make sure you have the following packages installed before compiling:

  • build-essential (includes GCC and Make)
  • libcrypt-dev (for password encryption support)
  • pkg-config (optional, if you plan to use external libraries)

To run the game server, navigate to the area directory and execute the binary with a desired port number: 

cd ../area
    ../src/rom 4000

Alternatively, you may use the startup.sh script located in the area directory to launch and log the server automatically: 

./startup.sh 4000

Once the server is running, you can connect to it using any MUD client such as Mudlet or CMUD, or simply by using telnet: 

telnet 127.0.0.1 4000

Enjoy building your own fantasy world!
