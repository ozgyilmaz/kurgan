Features

Explore the highlights of Kurgan MUD â€” a rebuilt and reimagined version of the legendary ROM 2.4b6

 

  • Based on ROM 2.4b6 with extensive modernization and cleanup
  • Text-based multiplayer fantasy role-playing game
  • Completely classless system with race-based mechanics
  • Randomized object generation (weapons, armor, potions, scrolls, etc.)
  • UTF-8 compatible input system (with support for Turkish characters)
  • 8-bit RGB color support with human-readable color names
  • JSON-based area file system with parser and converter
  • Clean and readable command output using printf_to_char() and act_color()
  • Book-based skill learning system with rarity mechanics
  • Dynamic quest system inspired by Anatolia and Vassago MUDs
  • Support for nonbinary, agender, and androgynous genders
  • Custom prompt configuration with room and exit coloring
  • Manuals and area data available in source repository
  • Designed for telnet clients or MUD clients like Mudlet
